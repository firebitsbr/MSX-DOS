BASIC
-----------------------------------------------------------
[ 機能 ]
BASIC を起動します。

[ 書式 ]
BASIC [<ファイル名>]

[ 文例 ]
<ファイル名> を入れると、BASIC 起動時にそのファイルが読み込まれ、実行されます。BASIC のみの場合は起動のみが行なわれます。
BASIC から MSX-DOS に戻る場合は CALL SYSTEM コマンド(_SYSTEM)を用います。
MSX-DOS と BASIC ではメモリ構成が行なっている為、メモリを介してデータのやりとりを行なう事はできません。ディスクに保存する等の対応が必要です。
